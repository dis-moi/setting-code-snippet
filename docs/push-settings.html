--- layout: snippet-loaded permalink: push-settings title: pushSettings( options, callback ) ---

Push options to the extension and call the callback function when the extension returns.

The callback is optional. options may contain:

  • subscriptions: [] a list of contributors identifiers to subscribe to.
{% highlight javascript %} const options = { // ... subscriptions: ['jdoe'], // ... }; lmem.pushSettings(options, () => console.log('LMEM extension set up!')); {% endhighlight %}
