--- layout: snippet-loaded permalink: on-install title: onInstall( callback ) ---

Call the callback function when the extension installation completes.

The callback is instantly called if the extension had been already installed.

{% highlight javascript %} lmem.onInstall(() => console.log('LMEM extension installed!')); {% endhighlight %}
